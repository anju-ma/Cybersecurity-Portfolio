# IPv4 (Internet Protocol Version 4)

## What is IPv4?

**IPv4 (Internet Protocol Version 4)** is the fourth version of the Internet Protocol. It is a **Layer 3 (Network Layer)** protocol that uses a **32-bit logical address** to identify devices and enable communication across IP networks.

**Example:**

```
192.168.1.10
```

---

## Why Do We Need IPv4?

IPv4 provides each device on an IP network with a unique logical address, allowing data to be delivered to the correct destination.

Without IPv4 addresses:

- Devices could not be uniquely identified on an IP network.
- Routers would not know where to forward data.
- Communication between networks and the Internet would not be possible.

---

## Structure of an IPv4 Address

An IPv4 address:

- Is **32 bits** long.
- Is divided into **4 octets**.
- Each octet contains **8 bits**.
- Is written in **dotted-decimal notation**.

**Example:**

```
192.168.1.10
```

```
192    .   168   .    1    .   10
│          │          │         │
Octet    Octet      Octet     Octet
```

Since each octet contains **8 bits**:

```
8 + 8 + 8 + 8 = 32 bits
```

Each octet has a decimal value between:

```
0 – 255
```

---

## Characteristics

- Uses a **32-bit logical address**.
- Consists of **4 octets**.
- Each octet contains **8 bits**.
- Written in dotted-decimal notation.
- Used to identify devices on IP networks.
- Supported by almost all networking devices.
- Provides approximately **4.3 billion** unique addresses.

---

## IPv4 Communication

Example:

```
Laptop IP  : 192.168.1.10

Printer IP : 192.168.1.20
```

When the laptop sends data, it places the printer's IPv4 address as the destination. The network forwards the packet so that it reaches the correct device.

## Advantages

- Simple and widely supported.
- Easy to configure.
- Compatible with almost all networking equipment.
- Well understood and widely used.
- Suitable for small and medium-sized networks.

---

## Disadvantages

- Limited address space (approximately 4.3 billion addresses).
- IPv4 addresses are becoming exhausted.
- Often requires **Network Address Translation (NAT)** because public IPv4 addresses are limited.
- Less scalable than IPv6.

---

## Real-World Example

Imagine sending a parcel through the postal service.

- **IPv4 Address** → House address
- **Router** → Postal sorting office
- **Packet** → Parcel

Without the correct address, the parcel cannot reach the correct destination.

---

## Cybersecurity Relevance

IPv4 is fundamental in cybersecurity because:

- Firewalls filter traffic using IP addresses.
- Security logs record source and destination IP addresses.
- SOC analysts investigate suspicious IP addresses.
- Security tools use IP addresses to monitor and analyze network activity.
- Incident responders use IP addresses to trace network connections.

---

## Key Takeaways

- IPv4 is the fourth version of the Internet Protocol.
- It operates at **OSI Layer 3 (Network Layer)**.
- It uses a **32-bit logical address**.
- An IPv4 address consists of **4 octets**.
- Each octet ranges from **0 to 255**.
- IPv4 is still the most widely used Internet Protocol.
- IPv6 was introduced to overcome the limitations of IPv4.

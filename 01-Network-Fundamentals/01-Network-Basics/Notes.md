# Network Basics

## 1. What is a Computer Network?

A computer network is a group of two or more devices connected together to communicate, exchange data, and share resources and services using wired or wireless communication.

## 3. How Devices Are Connected

Devices in a network communicate through a **communication medium**, which can be **guided (wired)** or **unguided (wireless)**.

### Wired (Guided)

Uses physical cables to transmit data.

**Examples:**
- Ethernet (Twisted Pair)
- Fiber Optic
- Coaxial

### Wireless (Unguided)

Uses electromagnetic waves instead of cables.

**Examples:**
- Wi-Fi
- Bluetooth
- Cellular (3G/4G/5G)
- Satellite

---

## 4. How Communication Happens

For two devices to communicate over a network:

1. Each device must have a **Network Interface Card (NIC)** or network interface to connect to the network.
2. The devices connect through a communication medium (wired or wireless).
3. Each device is identified by a **MAC address** on the local network and an **IP address** for communication across networks.
4. Network devices such as **switches** and **routers** forward data between devices.
5. Communication follows **network protocols** (such as TCP/IP), which define how data is formatted, transmitted, and received.

> **Note:** Network devices, IP addressing, and TCP/IP protocols are explained in detail in later sections of this repository.

## 5. Types of Networks

### PAN (Personal Area Network)

A Personal Area Network (PAN) connects personal devices over a very short range, typically within a few meters.

**Example:** A smartphone connected to a Bluetooth headset or smartwatch.

### LAN (Local Area Network)

A Local Area Network (LAN) connects devices within a limited geographical area.

**Examples:** Home, office, school, or college network.

### MAN (Metropolitan Area Network)

A Metropolitan Area Network (MAN) connects multiple LANs across a city or metropolitan area.

**Example:** A university connecting multiple campuses within the same city.

### WAN (Wide Area Network)

A Wide Area Network (WAN) connects multiple LANs over large geographical areas such as cities, states, countries, or continents.

**Example:** The Internet is the largest WAN in the world.

## 8. Real-World Example: Home Network

In a home network, devices such as a laptop, smartphone, Smart TV, and printer are connected to a Wi-Fi router. The router enables these devices to communicate with each other and access the Internet through an Internet Service Provider (ISP).

```text
                Internet
                    │
                   ISP
                    │
              Wi-Fi Router
        ┌────────┼────────┬────────┐
        │         │          │         │
     Laptop     Phone    Smart TV   Printer
```

## 9. Why Networking Is Important in Cybersecurity

Networking knowledge helps cybersecurity professionals to:

- Understand how data travels across a network
- Detect network attacks
- Analyze network traffic
- Secure network devices
- Understand IP addressing and network protocols
- Perform incident response
- Conduct digital forensic investigations
- Perform vulnerability assessments
- Monitor security events in a Security Operations Center (SOC)

---

## Key Takeaways

- A computer network connects devices for communication, data exchange, and resource sharing.
- Networks can use wired or wireless communication.
- Devices communicate using network interfaces, communication media, network devices, and protocols.
- PAN, LAN, MAN, and WAN are the main types of computer networks.
- Networking is a fundamental skill for cybersecurity professionals and SOC analysts.
